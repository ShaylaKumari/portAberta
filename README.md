# portAberta - Plataforma de Feedback Anônimo

portAberta é uma plataforma de feedback anônimo projetada para empresas que desejam criar um canal de comunicação seguro e transparente com seus colaboradores. A ferramenta permite que funcionários enviem feedbacks (elogios, sugestões, problemas) de forma anônima, e os gestores visualizem esses dados em um dashboard analítico para tomar decisões estratégicas.

### ✨ Funcionalidades Principais

| Funcionalidade | Descrição |
| --- | --- |
| **Feedback Anônimo** | Formulário público e simples para que qualquer colaborador possa enviar feedback sem se identificar. |
| **Dashboard Analítico** | Gestores têm acesso a um dashboard com métricas, gráficos e a lista de feedbacks recebidos. |
| **Segurança por Empresa** | Cada empresa tem acesso apenas aos seus próprios dados, garantindo a privacidade e o isolamento das informações. |
| **Relatórios Semanais** | Geração automática de relatórios em PDF com o resumo dos feedbacks da semana. |
